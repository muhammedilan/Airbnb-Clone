## Bilgisayarınızda Çalıştırın

Projeyi klonlayın

```bash
git clone https://github.com/muhammedilan/Airbnb-Clone.git
```

Proje dizinine gidin

```bash
cd Airbnb-Clone
```

Gerekli paketleri yükleyin

```bash
npm install
```

Sunucuyu çalıştırın

```bash
npm run start
```
